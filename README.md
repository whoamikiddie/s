# InnoSphere – Student Project Showcase

A modern React + Vite student project showcase website.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed via [Vercel](https://vercel.com). Push to `main` → auto-deploys.

### Custom Domain
Set up in Vercel Dashboard → Project → Settings → Domains.
